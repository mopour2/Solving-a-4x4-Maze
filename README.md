Solving a 4x4 Maze
The goal of this project is to develop a solution to solve 4x4 rectangular mazes with The Dobot Magician Lite robot's end effector must navigate through the maze in straight-line sections
To select the starting color of the gate, this system uses the ChatGPT5  agent's artificial intelligence and voice control method, and the agent prompt  is used for voice recognition, and for the response loop, an 8-step graph that controls all agents and actions is used.
This program is a synchronized program, considering the nature of libraries.
